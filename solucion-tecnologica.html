Entregable 3 — Diseño de interfaces

Pantallas del sistema propuesto

Wireframes y descripción de funcionamiento de las 5 pantallas que conforman el flujo automatizado de IPR.

El flujo de compra actual (catálogo → carrito → pago) ya existe en la web de IPR y no se modifica. Las pantallas descritas son las 5 nuevas interfaces que se activan a partir de que el cliente confirma el pago, cerrando el ciclo logístico integrado.

Registro y gestión de novedades post-despacho

🚚 Actor: Transportador → 💼 Asesor comercial

Pantalla 04 · Asesor comercial

Centro de Despacho y Entrega — Bandeja de pedidos especiales

💼 Actor: Asesor comercial
